# Arman Portfolio (Next.js + TypeScript + Tailwind + R3F)

This repository is a modern, futuristic personal portfolio scaffold for Shaik Arman Ali built with:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber (@react-three/fiber + @react-three/drei)

Setup

```
# install
npm install

# run dev
npm run dev
```

Notes

- Put your 3D avatar model at `public/avatar.glb`.
- Replace the placeholder text and projects with your own content.
- Add a `Shaik_Arman_Ali_CV.pdf` to `public/` to enable the CV download button.
